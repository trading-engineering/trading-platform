# Post-MVP Roadmap Boundaries

The following items are explicitly post-MVP and out of scope for the accepted
baseline:

- default flag flips
- GateDecision removal
- Strategy API redesign
- full order lifecycle model
- broad docs rewrite outside `core/docs`
- main docs repository cleanup

## Additional scope guard

Current MVP docs describe accepted transitional architecture and compatibility
bridges. They do not claim final-state completion.
